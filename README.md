### Installation

Clone repo:

```sh

git clone https://github.com/seraj/image-gallery.git

cd image-gallery

```

Install the dependencies:

```sh

yarn install

```

### Development Workflow

Start a live-reload development server:

```sh

yarn start

```

Generate a production build:

```sh

yarn build

```
